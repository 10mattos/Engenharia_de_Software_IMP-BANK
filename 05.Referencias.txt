05-Referencias 
Essa ideia surgiu na necessidade de uma integrante do grupo, na discussão das ideias ela nos contou um problema que ela teve para transferir seus dados para fazer um processo no Banco. E foi aí que surgiu a ideia de criar um sistema para intermediar documentos para um endereço de sistema e a partir dele você pode transferir seus dados para diversos tipos de instituições, como: Contratos de banco, inscrições para faculdade e todo tipo de necessidade e de transferência que pode existir. Visando hoje algumas plataformas têm a opção de upload, para anexar arquivo, porém em algumas plataformas não tem essa opção, já em outras eles pedem o arquivo depois existe um tempo para verificação desses documentos para aí sim dar continuidade ao processo, já por meio desse sistema os documentos já estariam aprovados agilizando a verificação das instituições diminuindo a mão de obra. O intuito e a referência que tivemos é a aceleração do andamento de dados de uma pessoa física para alguma instituição, por exemplo o caso de contratação de imóveis no cartório, usando a plataforma a pessoa física evitaria o incômodo de ir ao cartório para fazer o processo, isso daria pra ela e para o corretor uma flexibilidade.
